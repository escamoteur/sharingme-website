Verschlüsselung & Sicherheit

Wie wir Ihre Gedanken mit Ende-zu-Ende-Verschlüsselung schützen

Kritische Warnung: Verlieren Sie Ihr Passwort nicht

Wenn Sie als Kreisinhaber mit aktivierter Verschlüsselung den Zugriff auf Ihre App verlieren, kann das Passwort NICHT wiederhergestellt werden.

Alle verschlüsselten Daten im Kreis werden dauerhaft unzuganglich. Wir können Ihnen nicht helfen, verschlüsselte Daten ohne das Passwort wiederherzustellen.

Wir empfehlen dringend, Ihr Passwort in einem sicheren Passwort-Manager zu speichern.

Wie die Verschlüsselung funktioniert

Wenn Sie die Verschlüsselung füreinen Kreis aktivieren, verwendet Sharing Me Ende-zu-Ende-Verschlüsselung (E2EE), um Ihre Tagebucheintrage zu schützen. Das bedeutet:

  • Ihre Eintrage werden auf Ihrem Gerät verschlüsselt, bevor sie an unsere Server gesendet werden
  • Nur Personen mit dem Passwort können die Eintrage entschlüsseln und lesen
  • Unsere Server speichern nur verschlüsselte Daten - wir können Ihre Inhalte nicht lesen
  • Selbst wenn unsere Server kompromittiert wurden, bleiben Ihre Daten unlesbar

Verschlüsselung aktivieren

So aktivieren Sie die Verschlüsselung füreinen Kreis:

  1. Kreiseinstellungen offnen: Tippen Sie auf den Kreisnamen oder gehen Sie zu "Meine Kreise" und tippen Sie auf das Bearbeiten-Symbol des Kreises, den Sie verschlusseln mochten.
  2. Verschlüsselung aktivieren: Schalten Sie den Schalter "Verschlüsselung aktiviert" auf EIN.
  3. Passwort speichern: Ein eindeutiges Passwort wird generiert. Speichern Sie dieses Passwort sicher!

Passwort anzeigen & kopieren

Wenn Sie Ihr bestehendes Passwort anzeigen oder teilen müssen:

  1. Gehen Sie zu den Einstellungen Ihres Kreises (tippen Sie auf den Kreisnamen oder das Bearbeiten-Symbol)
  2. Suchen Sie den Schalter "Verschlüsselung aktiviert"
  3. Zum Anzeigen: Tippen Sie auf das Auge-Symbol , um das Passwort anzuzeigen
  4. Zum Kopieren: Tippen Sie auf das Kopieren-Symbol , um das Passwort in die Zwischenablage zu kopieren

Mit Kreismitgliedern teilen

Wenn Sie jemanden in einen verschlüsselten Kreis einladen, benotigt diese Person das Passwort, um Eintrage zu lesen. Sie sollten:

  • Das Passwort über einen sicheren Kanal teilen (personlich, verschlüsselte Nachrichtenapps usw.)
  • Vermeiden, das Passwort über unverschlüsselte E-Mail oder SMS zu senden
  • Erwagen, einen Passwort-Manager mit sicherer Freigabefunktion zu nutzen

Jedes Kreismitglied gibt das Passwort einmalig in seiner App ein. Es wird sicher auf dem Gerät fürzukunftige Verwendung gespeichert.

Was passiert, wenn Sie das Passwort verlieren

Wichtig: Keine Wiederherstellungsoption

Da wir echte Ende-zu-Ende-Verschlüsselung verwenden, haben wir keinen Zugriff auf Ihr Passwort und können es nicht zurücksetzen. Wenn Sie das Passwort verlieren:

  • Können Sie bestehende verschlüsselte Eintrage nicht lesen
  • Andere Kreismitglieder mit dem Passwort haben weiterhin Zugriff auf die Daten
  • Sie können die Verschlüsselung deaktivieren, aber bestehende verschlüsselte Eintrage bleiben unlesbar
  • Neue Eintrage (nach Deaktivierung) werden nicht verschlüsselt

Empfehlungen

Passwort-Manager verwenden

Speichern Sie Ihr Passwort in einem seriosen Passwort-Manager wie 1Password, Bitwarden oder Apple Schlüsselund.

Backup aufbewahren

Schreiben Sie das Passwort auf und bewahren Sie es an einem sicheren Ort auf (Bankschließfach, feuerfester Safe usw.).

Sicher teilen

Teilen Sie das Passwort beim Einladen von Mitgliedern personlich oder über eine verschlüsselte Nachrichtenapp.

Technische Details

Fur Interessierte an der technischen Umsetzung:

  • Algorithmus: AES-256-GCM Verschlüsselung
  • Schlüsselableitung: PBKDF2 mit eindeutigem Salt pro Kreis
  • Passwort: Kryptografisch zufallig, clientseitig generiert
  • Speicherung: Passwort im sicheren Gerätespeicher (Keychain/Keystore)
